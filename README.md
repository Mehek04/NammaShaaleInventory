# 📚 NammaShaaleInventory

> *Smart School Asset & Inventory Management System for Efficient Monitoring and Maintenance*

---

## ✨ Overview

**NammaShaaleInventory** is an Android-based Inventory and Asset Management application developed to help schools efficiently manage, monitor, and maintain institutional assets through a structured digital platform.

The application simplifies asset tracking, repair request handling, issue reporting, and monthly health monitoring using a clean, responsive, and user-friendly interface. The project focuses on improving organization, reducing manual management efforts, and ensuring efficient maintenance of school resources.

---

## 🚀 Key Features

✅ Asset Management System
✅ Monthly Asset Health Check
✅ Repair Request Tracking
✅ Issue Logging & Monitoring
✅ Asset Status Management
✅ Search & Filter Functionality
✅ Offline Data Storage using Room Database
✅ MVVM Architecture Implementation
✅ Modern Material Design UI

## 🏗️ System Architecture

The project follows the **MVVM (Model–View–ViewModel)** architecture to maintain clean code structure and improve scalability.

* **View** → User Interface Layer
* **ViewModel** → Business Logic Layer
* **Model/Data Layer** → Data Handling Layer
* **Room Database** → Local Data Persistence
