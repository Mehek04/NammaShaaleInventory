\# NammaShaaleInventory



NammaShaaleInventory is an Android application for managing school inventory, assets, repairs, issue logs, and reports.



\## Features



\- Asset management

\- Issue log tracking

\- Repair request tracking

\- Monthly health checks

\- Dashboard and reports

\- QR code support

\- CSV/PDF export



\## Tech Stack



\- Kotlin

\- Android Studio

\- Gradle

\- Room Database
